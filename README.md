# Ander-Labs Portfolio

Bilingual EN/ES portfolio for Anderson Ruiz.
Built with Astro and a minimal dark theme.

## Stack
- Astro 7
- Static site, no runtime dependencies
- Bilingual routing: `/` EN, `/es/` ES

## Sections
- Home
- About
- Projects
- Blog
- Contact

## Development
```sh
npm install
npm run dev
```

## Environment
Copy `.env.example` to `.env` and set `PUBLIC_SITE`:
```sh
cp .env.example .env
```
